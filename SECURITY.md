# Security Policy

`nihongo-slopless` は文章 lint 用のCLIです。秘密情報、非公開文書、学生情報、組織内文書を含む入力を、公開ログ、公開SARIF、公開Issue、PRコメントに貼らないでください。

## 報告してほしいもの

- 除外範囲、コードブロック、HTMLコメント、front matter、inline code、disable範囲の本文が `excerpt` に漏れる問題。
- ローカル絶対パスや非公開ファイル名が、既定出力や公開用SARIFに混ざる問題。
- npmパッケージに含めるべきでないファイルが含まれる問題。
- `ignorePatterns` や無視コメントが、想定より広く本文を露出または抑制する問題。

## 報告先

GitHub Issuesを使ってください。ただし、秘密情報を含む再現例は貼らず、最小化したダミー例で報告してください。

実際の鍵、トークン、学生情報、組織内URL、非公開ファイルパスが出力に混ざった場合は、公開Issueに貼る前に該当情報を無効化または伏せ字化してください。
